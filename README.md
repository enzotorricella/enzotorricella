## Enfoque

Construyo y mantengo aplicaciones de negocio con C#, .NET y SQL. Me interesan los sistemas que tienen uso operativo real: eventos, comercios, soporte de aplicaciones, datos y herramientas internas.

Trabajo especialmente sobre:

- aplicaciones .NET, APIs y servicios backend;
- consultas SQL, depuracion de datos y soporte productivo;
- aplicaciones desktop/mobile con .NET MAUI, WPF y Windows Forms;
- integraciones, reglas de negocio y flujos operativos;
- infraestructura basica para despliegue, diagnostico y mantenimiento.
